## Feedback para Ações

- **Ataque:** Animação + som de impacto + número de dano flutuante
- **Dano Recebido:** Tela pisca vermelho + som de impacto + número de dano
- **Interação:** Highlight do objeto + som de interação + prompt visual
- **Habilidade:** Animação + efeito visual + som característico
- **Level Up:** Efeito visual + som de conquista + notificação
- **Quest Completa:** Notificação + som de sucesso + recompensa visual

## Cores e Estilo

- **HP:** Vermelho (quando baixo), verde (quando alto)
- **Mana:** Azul
- **XP:** Amarelo/Dourado
- **Dano:** Vermelho (inimigo), amarelo (crítico)
- **Cura:** Verde

## Acessibilidade

- Opções de tamanho de texto
- Indicadores visuais além de cores (para daltônicos)
- Opções de contraste
