## Mapeamento de Controles (PC - Teclado e Mouse)

| Ação | Tecla/Botão | Resposta do Sistema |
|------|-------------|---------------------|
| **Movimentar** | WASD | Move personagem na direção pressionada |
| **Atacar** | Botão Esquerdo do Mouse | Executa ataque corpo a corpo ou à distância (baseado em arma equipada) |
| **Interagir** | E | Interage com objetos/NPCs dentro do raio de interação |
| **Usar Habilidade 1** | 1 | Ativa primeira habilidade da hotbar |
| **Usar Habilidade 2** | 2 | Ativa segunda habilidade da hotbar |
| **Usar Habilidade 3** | 3 | Ativa terceira habilidade da hotbar |
| **Usar Habilidade 4** | 4 | Ativa quarta habilidade da hotbar |
| **Abrir Inventário** | I | Abre/fecha menu de inventário |
| **Abrir Ficha** | C | Abre/fecha ficha de personagem |
| **Abrir Mapa** | M | Abre/fecha mapa do mundo |
| **Pausar** | ESC | Pausa o jogo e abre menu principal |

## Mapeamento de Controles (Mobile - Touch)

| Ação | Input | Resposta do Sistema |
|------|-------|---------------------|
| **Movimentar** | Analógico Virtual (esquerda) | Move personagem na direção do analógico |
| **Atacar** | Botão de Ataque (direita) | Executa ataque corpo a corpo ou à distância |
| **Interagir** | Botão de Interação | Interage com objetos/NPCs dentro do raio |
| **Usar Habilidades** | Hotbar (inferior) | Toque na habilidade para ativar |
| **Abrir Menus** | Botões de Menu | Abre/fecha menus (inventário, ficha, mapa) |

## Feedback Visual e Sonoro

- **Ataque:** Animação de ataque + som de impacto + feedback visual (dano flutuante)
- **Interação:** Highlight do objeto + som de interação + prompt visual
- **Habilidade:** Animação + efeito visual + som característico
- **Dano Recebido:** Tela pisca vermelho + som de impacto + número de dano
