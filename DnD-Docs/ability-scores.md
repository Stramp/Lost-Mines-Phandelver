# Ability Scores

## Ability Names

- Strength (STR)
- Dexterity (DEX)
- Constitution (CON)
- Intelligence (INT)
- Wisdom (WIS)
- Charisma (CHA)

## Modifier Calculation

```
Modifier = floor((Score - 10) / 2)
```

## Score Ranges

- Base (Point Buy): 8-15
- After Racial Bonuses: 8-17 (typically)
- Maximum (with ASI): 20

## Final Score Formula

```
FinalScore = BaseScore + RacialBonuses + ASI
```

## Ability Score Improvements (ASI)

- Classes get ASI at levels: 4, 8, 12, 16, 19
- +2 to one ability OR +1 to two abilities
- Fighter/Rogue get extra ASI at levels 6, 10, 14
