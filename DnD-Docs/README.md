- `point-buy-system.md` - Sistema de distribuição de pontos (27 pontos)
- `ability-scores.md` - Ability scores e modificadores
- `multiclassing.md` - Regras de multi-classing

Referência rápida para implementar lógica de validação e cálculo no projeto.
