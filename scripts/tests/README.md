# Scripts de Teste

Esta pasta contém todos os scripts relacionados a testes do projeto.

## Estrutura

```
scripts/tests/
├── README.md (este arquivo)
└── [scripts de teste aqui]
```

## Scripts de Teste

Scripts relacionados a testes do Unreal Engine e automação de testes devem ser colocados nesta pasta.

## Exemplos

- Scripts para rodar testes do Unreal Engine
- Scripts para validar resultados de testes
- Scripts para gerar relatórios de testes
- Scripts auxiliares para testes
